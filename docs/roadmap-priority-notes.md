本文件用于保存 `roadmap.md` 的继续开发优先级，避免后续忘记哪些功能已经半落地、哪些值得先收口。

## 第一梯队

1. 全局提示词大编辑器第二阶段
   - 已完成核心节点接入。
   - 继续做配置型长文本：ApiSettings、ComfyUI 应用制作工具、RH 工具箱制作器、RH 工具编辑弹窗。
   - 重点能力：大编辑、JSON 格式化/校验、行列表整理、个人偏好预留。

2. 3D 全景节点体验收口
   - 已有 21:9、1K/2K、GPT Image 2 文生/图生全景、生成历史、资源库保存。
   - 继续做：接缝质量提示、极点/比例提示、全景贴图与当前视角导出区分、失败保留旧图、资源库 metadata。

3. 主题成就 / 有效时长系统
   - 已完成时长、勋章、隐藏模式、影片占位。
   - 继续做：资源库保存事件、工作流保存事件、导入 UI、今日主题任务、本地创作回顾。
   - 奖励影片播放/加密等用户提供素材后再开发。

4. 聚合解析 Cookie 授权助手
   - 已有平台识别、Electron 授权窗口、本机加密授权库、结构化错误和默认下载模式。
   - 继续做：更清晰的一步步授权向导、授权窗口打开后自动提示检测、失败时给平台级下一步、减少用户对 Cookie 的理解负担。

## 第二梯队

1. ComfyUI 本地 API 简化接入 + ComfyUI 超市/应用制作工具
   - 继续做样例工作流、字段识别准确率、导入向导和缺模型/缺节点错误提示。
   - 2026-06-04 已收口：内置基础文生图样例、设置页/制作工具「载入样例」、导入检查清单、超市运行前素材缺口提示、后端缺模型/缺自定义节点/未启动/校验失败错误分类。
   - 后续继续做：ControlNet / OpenPose / Mask / Video 专用映射来源，以及从 ComfyUI 历史任务导入当前 prompt/workflow。

2. RH 工具箱
   - 继续把能力快捷入口接入图像、视频、文本、音频节点，例如抠图、放大、扩图、视频放大、文本扩写。
   - 2026-06-04 已补底座：`buildRhToolboxQuickActions()` 按图像/视频/文本/音频 surface 生成快捷入口矩阵，RH 工具箱节点显示“快捷接入位”；后续图像/视频/文本/音频节点只需复用该 helper 执行具体按钮接入。

3. 云端上传目标
   - 先把 COS/OSS 做稳。
   - 2026-06-04 已补可诊断性：`classifyCloudUploadError()` 将 COS/OSS 的签名、密钥、Bucket/Region/Endpoint、权限和网络错误转成可执行中文提示，上传路由返回结构化 `code/hint`。
   - 百度网盘/夸克继续保留实验或占位，不急着承诺稳定上传。
