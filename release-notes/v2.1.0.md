# v2.1.0 · Prompt media templates and Remote ComfyUI source release

## Highlights

- Upgraded the prompt template library from text-only entries to media-aware templates for image, video, audio, and text materials.
- Added right-click save-to-template flows for generated/uploaded/output materials, preserving the prompt, title, tags, media type, and matching media attachments.
- Added lazy media preview behavior in the template library, including hover eye controls for 100% image inspection without loading every full asset at once.
- Added a Remote ComfyUI high-risk switch that stays off by default. Localhost remains the default policy, while trusted remote endpoints can be enabled per ComfyUI config with `allowRemote`, or globally through `T8_COMFYUI_ALLOW_REMOTE=1` / `T8_COMFYUI_ALLOW_PRIVATE=1`.
- Added Web + backend Docker deployment files: `Dockerfile`, `docker-compose.yml`, and `.dockerignore`. Docker deployment does not include Electron.
- Centralized ComfyUI URL access checks in `backend/src/providers/comfyuiAccess.js` so provider normalization, test connections, and real runs share the same safety policy.

## Source-Only Boundary

- This version is a source-only GitHub update.
- No Electron installer was built.
- No GitHub Release or `latest.yml` auto-update asset was created or updated.
- Public source continues to exclude `skill.md`, `roadmap.md`, `local-private/**`, and local-only commercial overlay code.
