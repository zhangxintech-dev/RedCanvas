# v2.1.3 Source Update

发布日期：2026-06-08

本次为 GitHub 源码更新。按当前发布边界不打包 Electron，不运行 `npm run dist:release`，不创建或更新 GitHub Release / `latest.yml` 自动更新资产。

## 更新内容

- 按用户提供的方式更新本机即梦 CLI：`curl -fsSL https://jimeng.jianying.com/cli | bash`，新版安装到 `C:\Users\Administrator\bin\dreamina.exe`。
- 新版安装包没有单独提供 `tools.json`；安装后的 `SKILL.md` 明确要求以 `dreamina -h` 和各子命令 `-h` 作为模型支持的权威来源。
- `dreamina text2image -h` 与 `dreamina image2image -h` 已确认支持 `model_version=4.7`，因此即梦 CLI 图像模型新增 `seedream-4.7`，后端提交时会写入 `--model_version=4.7`。
- 即梦 CLI 默认图像模型列表同步补充 `seedream-4.7 / seedream-4.6 / seedream-4.5 / seedream-5.0`，旧的 `jimeng-image-2k / jimeng-image-4k` 保留兼容。
- `dreamina text2video -h` 与 `dreamina multimodal2video -h` 已确认 Seedance 2.0 视频模型支持 `seedance2.0fast_vip / seedance2.0_vip / seedance2.0fast / seedance2.0`，默认列表和旧设置迁移已补齐。
- 按 CLI 帮助同步 Seedance 2.0 分辨率边界：只有 `seedance2.0_vip` 允许 1080p，其余 Seedance 2.0 变体提交时会固定为 720p，避免 CLI 参数报错。
- API Key 设置页即梦 CLI 模型提示同步更新，方便用户直接看到 `seedream-4.7` 和完整 Seedance 2.0 变体。

## 边界说明

- 本次没有提交 `skill.md`、`roadmap.md`、`local-private/**` 或任何本地私有实现。
- 本次只做源码更新；Electron 用户自动更新包仍需后续单独按 release 流程打包发布。
