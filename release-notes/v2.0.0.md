# 贞贞的无限画布 v2.0.0

- 固化画布性能规范：新增节点图片预览必须优先使用 SmartImage / previewImageUrl，本地 input/output 图片走后端 webp 缩略图缓存。
- 优化大画布交互：体育主题连线动画在拖动画布、拖动节点或大边数场景自动降级，减少持续动画、filter 和运动球开销。
- 优化开发体验：开发环境默认关闭 React StrictMode 双挂载，可通过 VITE_T8_STRICT_MODE=1 恢复诊断模式。
- 降低拖动卡顿：拖动节点期间暂停中间帧自动保存，拖动结束后补最终快照。
- 规范版本进位：三段版本号每段只允许 0-9，到 9 后必须向左进位；本版从 v1.9.9 进位到 v2.0.0。
- Release 资产包含 NSIS 安装包、blockmap 与 latest.yml，可通过 GitHub Release 自动更新。
