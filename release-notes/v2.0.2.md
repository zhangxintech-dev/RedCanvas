# 贞贞的无限画布 v2.0.2

- 聚合解析授权助手区分浏览器手动模式与 Electron 桌面自动模式，避免普通浏览器环境误以为可自动读取平台 Cookie。
- 浏览器模式下按钮改为“打开平台官网 / 检查已粘贴 Cookie”，并明确提示 `127.0.0.1` 页面不能跨站读取 Cookie。
- Electron 桌面端继续支持隔离授权窗口、检测授权、加密保存本机授权库、载入授权和清除 Cookie。
- 保留开发首屏懒加载、图片缩略图队列、3D 全景资源库和全局提示词大编辑器等 v2.0.1 源码能力。
- Release 资产包含 NSIS 安装包、blockmap 与 `latest.yml`，供桌面端自动更新使用。
