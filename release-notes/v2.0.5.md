# v2.0.5 · roadmap continuation source release

This is a source-only release. Electron packaging, installer assets, `latest.yml`, and GitHub Release updater assets were intentionally not produced for this version.

## Highlights

- Added the first version of the theme achievement and active-time system, including shared rules, backend profile/event APIs, frontend tracker, drawer, button, toast, import/export, and placeholder film rewards.
- Expanded the global large prompt editor with text, JSON, and line-list modes, then wired it into API settings, ComfyUI app authoring, and RH toolbox editors.
- Added lightweight 3D panorama texture quality hints and achievement events for panorama generation, resource saves, and workflow saves.
- Improved the aggregate parser authorization guide with a clearer step-by-step flow for browser and desktop usage.
- Added ComfyUI starter workflow samples, import checklists, and friendlier diagnostics for missing models, missing custom nodes, offline ComfyUI, validation failures, and timeouts.
- Added RH toolbox quick action metadata for image, video, text, and audio surfaces.
- Added cloud upload error classification and actionable hints for common COS/OSS configuration, permission, endpoint, and network failures.
- Added grid editor captions, caption styling controls, and backend `grid-compose` caption rendering.

## Verification

- `node --test tests\*.test.ts`
- `npm run type-check`
- `npm run build`

## Packaging

No Electron package or updater release was published for v2.0.5. The next desktop update must be built from a clean public worktree with the normal `npm run dist:release` workflow.
