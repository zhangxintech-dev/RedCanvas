# v2.1.5 Source Update

发布日期：2026-06-09

本次为 GitHub 源码更新。按当前发布边界不打包 Electron，不运行 `npm run dist:release`，不创建或更新 GitHub Release / `latest.yml` 自动更新资产。

## 更新内容

- 新增 Fal 超市公开节点，复用通用 FAL manifest / submit / query 链路，把已核验的 Fal.ai 模型能力按分类组织；原图像、视频等节点内已有 FAL 能力保持不变。
- 新增 3D 素材上传、3D 模型预览与 OutputNode 3D 模型分区；FAL 3D 产物不再按图片渲染，后续会自动接到 3D 预览节点。
- 修复 3D 模型预览 removeChild 崩溃、预览节点无限增殖和快照输出问题。
- RH 工具箱运行端改为轻量双列按钮，悬停工具时显示工具说明；Prompt/text 字段会直接显示模板默认值，用户不需要重新手打。
- RH 工具箱运行端会按 inputSchema 直显 Prompt、图像、视频、音频字段，文本字段接入提示词模板、放大编辑和 @ 素材引用。
- 新增 Grok OAuth Agent 公开外壳和 hook 路由；真实 OAuth 登录、token 刷新和 Grok / xAI 调用仍由本地私有扩展提供。
- 新增长按鼠标中键的圆盘菜单及自定义入口，并修复中键圆盘打开时画布抖动。
- 新增圣斗士主题首版：圣衣收集、宝箱、小地图、战斗、经验升级、冥界篇隐藏模式与成就占位。
- `public:check` 增加 RH 工具箱制作器和 FAL 应用制作工具源码拦截，避免维护者开发工具误入公开提交。

## 边界说明

- 本次没有提交 `skill.md`、`roadmap.md`、`local-private/**`、`hermes-agent/**` 或 Electron 缓存目录。
- `src/components/nodes/RHToolboxMakerNode.tsx` 与 `src/components/nodes/FalToolboxMakerNode.tsx` 是维护者本地开发工具，本次不提交源码到 GitHub。
