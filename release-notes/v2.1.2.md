# v2.1.2 Source Update

发布日期：2026-06-08

本次为 GitHub 源码更新。按用户要求不打包 Electron，不运行 `npm run dist:release`，不创建或更新 GitHub Release / `latest.yml` 自动更新资产。

## 更新内容

- LLM 节点支持删除单条 assistant 流式结果，删除后会同步刷新输出字段，后续生成不会让旧结果重新出现。
- 提示词 `@` 提及修复即时弹出时机，文本节点也支持连接上游图像 / 视频 / 音频后进行素材提及预览。
- 文本分割节点输入框接入统一提示词模板库和放大编辑器。
- 大画布框选结束时按实际屏幕拖拽矩形复核选区，降低节点较多时框选漂移误选远处节点的概率。
- 组框右键菜单新增“删除组和内容”，可删除组框及几何范围 / memberIds 内的节点，并清理相关连线。
- 素材右键保存到提示词模板库时可选择模板分类，也可就地新建模板分类；模板库本身继续支持分类新增、重命名和删除。
- 发送弹窗新增发送到 Figma，本机桥接接口只允许 localhost，避免把本地素材发送到远端。
- 新增左下角放置栏：发送、粘贴和自动输出的素材节点会登记映射，折叠显示最近 5 个，展开显示最近 20 个；拖动放置栏素材会移动原节点到落点，不会复制节点。
- 图像 / 视频 / 音频素材元素写入浏览器 `DownloadURL` / `text/uri-list` 拖拽数据，支持直接拖到浏览器外文件夹。
- RunningHub 上传转换链路增加回归检查，确认 `/files/input/*` 上传节点素材与 `/files/output/*` 输出素材都可走 `/runninghub/upload-asset` 转 RH fileName。
