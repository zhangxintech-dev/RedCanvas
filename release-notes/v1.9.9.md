# 贞贞的无限画布 v1.9.9

- 修复 3D 全景节点在开发环境从 `/public/vendor/js` 动态导入 Three.js 导致 Vite 白屏 overlay 的问题，改为随应用依赖按需加载 `three`。
- 3D 全景自动旋转只在贴图就绪且用户开启时运行，减少空画布/未就绪节点的首屏渲染压力。
- 上传、输出、上游素材缩略图和 3D 全景导出预览改为 lazy loading + async decoding，降低大画布启动时图片抢占网络与解码资源的问题。
- v1.9.9 起更新下载完成后的“重启安装”改为静默安装并自动重启新版，减少后续版本更新时弹出安装向导的打断。
- 本版继续保留 GitHub Release 自动更新链路，用作从 v1.9.8 基线包升级验证的新版本。
